# 番茄钟 🍅

一个纯前端番茄工作法计时器，双击 `pomodoro.html` 即可使用，无需安装任何依赖。

## 功能

- **三种模式**：专注（25 分钟）→ 短休（5 分钟）→ 长休（15 分钟）
- **自动切换**：专注结束自动进入休息，休息结束自动回到专注；每 4 轮专注后进入长休
- **环形进度条**：直观显示剩余时间
- **开始 / 暂停 / 重置**：随时控制计时
- **提示音 + 桌面通知**：计时结束时提醒
- **自定义时长**：点击右上角齿轮图标，可修改三种模式的时长，自动保存到浏览器
- **键盘快捷键**：
  - `空格` — 开始 / 暂停
  - `R` — 重置
  - `1` / `2` / `3` — 切换到专注 / 短休 / 长休

## 使用方式

双击 `pomodoro.html`，在浏览器中打开即可。
